# CCO-Samples

To "clean up" the data files you can use ant (with build.xml for "data_old" or build_new.xml for "data_new") with the default target or run the xslt-files in the following order:

1. cleanup-basic.xsl
1. cleanup-types.xsl
1. cleanup-source.xsl
1. cleanup-sort.xsl
1. cleanup-dates.xsl
1. cleanup-sets.xsl
1. cleanup-empty.xsl
1. cleanup-basic.xsl
